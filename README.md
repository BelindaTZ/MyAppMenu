# MyAppMenu 📱

Aplicación de menú interactivo desarrollada en **Kotlin** para Android. Una interfaz elegante y funcional que permite navegar entre diferentes secciones de la aplicación.

## 🎯 Características

- **Interfaz moderna**: Diseño atractivo con gradientes de colores
- **Navegación intuitiva**: Menú organizado y fácil de usar
- **Gestión de datos**: Secciones para Clientes, Productos y Proveedores
- **Opciones adicionales**: Módulos flexibles para extensión

## 📸 Captura de Pantalla

![MyAppMenu](https://github.com/BelindaTZ/MyAppMenu/raw/main/screenshot.png)

La aplicación presenta las siguientes secciones principales:

- 👥 **Clientes**: Gestión de información de clientes
- 🛒 **Productos**: Catálogo y administración de productos
- 🏢 **Proveedores**: Información de proveedores (destacada)
- ⚙️ **Otras Opciones**: Menús adicionales configurables

## 🛠️ Tecnologías

- **Lenguaje**: Kotlin 100%
- **Plataforma**: Android
- **Arquitectura**: Componentes de UI modernos

## 📋 Requisitos

- Android Studio
- SDK mínimo: Android 5.0 o superior
- Kotlin 1.5+

## 🚀 Inicio Rápido

1. Clona el repositorio:
```bash
git clone https://github.com/BelindaTZ/MyAppMenu.git
```

2. Abre el proyecto en Android Studio

3. Sincroniza las dependencias de Gradle

4. Ejecuta la aplicación en un emulador o dispositivo físico
